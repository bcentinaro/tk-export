# TkExport

This tool requires Elixir be installed. See https://elixir-lang.org/install.html for more details. 

## Setup

`mix deps.get` 

## Export 

The Export is depends on two variables USER_ID and TK_COOKIE

***Mac/Linux (Bash)***
`USER_ID=<id> TK_COOKIE=<cookie> mix export`
